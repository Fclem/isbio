Introduction

The main idea behind BREEZE project is to build a 24/7 communication bridge between biologists and bioinformatics resources.

Overview

Development status: BETA VERSION

Breeze sources

  • Breeze Cloud source code on GitHub
  • FIMM's Breeze source code on GitHub
  • Breeze's Docker infrastructure sources on GitHub : includes MySql schemas and managements shell scripts and docker build files for Django, Breeze, Nginx.
  • R execution environnement build sources on GitHub : for R2 and R3 environnements.
  • Custom docker building system build sources on GitHub : this builder was created to simplify the building of the various R environements.

Technologies used in Breeze :

    System, and infrastructure
  • Python 2.7.11, programing language
  • Django 1.9.6, Python web framework
  • MySql, SQL Database
  • Docker, software containerization, i.e. lightweight virtualization
  • Nginx, web server
  • Ubuntu, Linux-Debian based Operating System
  • CentOS, Linux-RedHat based Operating System
  • Microsoft Azure, PaaS virtualization services
  • OpenSSH, encryption, virtual networking, tunneling, etc
  • CSC's Pouta cloud service, PaaS virtualization services
  • R 2.15.3 (Security Blanket), R 3.2.1 (World-Famous Astronaut), R 3.3.2 (Sincere Pumpkin Patch), statistical and bioinformatics programming language
    Services
  • Docker HUB, Docker container public repositories
  • Auth0, OAuth 3rd party delegated authentication
  • GitHub, online source code repository for git versioning tool
  • Rollbar, automated bug tracking
  • DataDog, infrastructure tracking, analysis and monitoring
  • Zendesk Chat (Zopim), end-user live-chat for user-support
    Front-end
  • Bootstrap, Bootstrap Tour
  • Glyphicons
  • Shiny, Shinydashboard
  • jQuery, jQuery-UI
